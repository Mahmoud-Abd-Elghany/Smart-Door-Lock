# Smart-Door-Lock
### Consists of 2 ECUs: 
* First ECU (HMI) responsible for Interfacing with the User.
* Second ECU responsible for System operations and Control.
### Drivers Implemented:
- UART(Communication between 2 uC).
- SPI (Connection between uC and EEPROM)
- External EEPROM
- LCD
- Keypad
